Qt实现 登录注册界面

理解：

qt实现登录注册界面，本质是数据库的查询和增加操作，同时也用到Qt connect()函数!!!

项目文件夹 如图：

1.连好数据库， 建好关系表

mydatabase.h 有个 class MyDatabase ,实现对数据库的 封装保护

mydatabase.cpp 里面有 void init()函数， 实现数据库的连接 和 建好关系表

2.分别实现登录、注册界面

//登录（widget窗体 登录成功后 跳转到 mainwindow窗体）

特点：两个窗体之间的 跳转

main.cpp里面 有

核心一：QObject::connect(&w, SIGNAL(sigLoginSuccess(QString)), &mw, SLOT(onLogin(QString)));

class Widget的构造函数里面

核心二：connect(ui->btn_onLongin, SIGNAL(clicked(bool) ),

this, SLOT(onLogin()) );

实现点击登录界面(widget窗体)的 登录按钮后， 跳转到 主界面(mainwindow窗体)。

①ui->btn_onLongin：点击登录按钮，发送信号。

mydatabase.cpp 里面有 MyLogin()实现数据库数据的 查询

②widget.cpp 里面的 onLogin()函数基于上面的MyLogin()函数 去响应实现 登录按钮的点击事件

③mainwindow.cpp里面有 onLogin()槽函数， 实现显示 主界面(maindow窗体)
